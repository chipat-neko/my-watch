Libre & open source

Clone le dépôt, ajoute tes clés, et lance l'app avec Expo Go.

# Récupérer et lancer
git clone https://github.com/chipat-neko/my-watch.git
cd my-watch
npm install
npm start

Projet indépendant · GitHub

Ce produit utilise l'API de TMDb mais n'est ni approuvé ni certifié par TMDb. Les données de films et séries proviennent de The Movie Database (TMDb). My Watch est un projet indépendant et ne reprend ni le code, ni les logos, ni les marques de TV Time.
